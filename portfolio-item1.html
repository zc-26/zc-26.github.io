Java开发

Java图书管理系统

开发时间

2024年10月-11月

技术栈

Java Swing MySQL

开发工具

IntelliJ IDEA、Navicat、MySQL

项目简介

基于Java Swing开发的桌面版图书管理系统，实现图书借阅、归还、查询、用户管理等核心功能，适用于小型图书馆或书店。

核心功能

  • • 图书信息管理（添加、修改、删除、查询）
  • • 用户管理（注册、登录、权限控制）
  • • 借阅/归还管理（记录借阅时间、自动计算归还日期）
  • • 数据统计（图书借阅排行、用户借阅记录）
